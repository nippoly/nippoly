![logo](./images/nippo_logo.png)

## What is nippoly  

**「読んだ！が10倍速くなる」** を実現する24時間以内に読んだ記事をまとめてマークダウン形式で出力してくれるChrome Extensionです。


## How to set up Developer mode
### 手順
1. **ツール > 拡張機能** にアクセス
2. **デベロッパーモード > パッケージ化されていない拡張機能を読み込む** を選択肢します
3. **chrome_extension** というフォルダを選択します
